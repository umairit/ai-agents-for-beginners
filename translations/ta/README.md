>
> இந்த சேமிப்பகத்தில் 50+ மொழி மொழிபெயர்ப்புகள் உள்ளன, இது பதிவிறக்க அளவை முக்கியமாக அதிகரிக்கிறது. மொழிபெயர்ப்புகளின்றி நகலை நோக்க sparse checkout பயன்படுத்தவும்:
>
> **Bash / macOS / Linux:**
> ```bash
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
>
> **CMD (Windows):**
> ```cmd
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"
> ```
>
> இது நீங்கள் பாடநெறியை முடிக்க விரைவான பதிவிறக்கத்துடன் தேவையான அனைத்தையும் வழங்கும்.

## 🌱 தொடக்கம்

இந்த பாடநெறியில் செயற்கை நுண்ணறிவு முகவர்களை கட்டமைப்பதற்கான அடிப்படைகளை உள்ளடக்கிய பாடங்கள் உள்ளன. ஒவ்வொரு பாடமும் தனக்கு சம்பந்தப்பட்ட தலைப்பை உள்ளடக்குகிறது, எனவே விரும்பிய இடத்தில் இருந்து தொடங்குங்கள்!

இந்த பாடநெறிக்கு பன்மொழி ஆதரவு உள்ளது. எங்கள் [கிடைக்கும் மொழிகளுக்குச் செல்லவும்](#-multi-language-support).

இது நீங்கள் முதன் முறையாக ஜெனரேட்டிவ் AI மாடல்களுடன் கட்டமைப்பதாக இருந்தால், 21 பாடங்களை கொண்டுள்ள எங்கள் [தொடக்கத்தரவையர் க்ஹானியற்கான ஜெனரேட்டிவ் AI](https://aka.ms/genai-beginners) பாடநெறியை சரிபார்க்கவும்.

இந்த சேமிப்பகத்தை [நட்சத்திரம் (🌟) செய்யவும்](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) மற்றும் [இந்த சேமிப்பகத்தை கிளோன் செய்யவும்](https://github.com/microsoft/ai-agents-for-beginners/fork) கோட் இயக்க இதை மறக்காதீர்கள்.

### மற்ற கற்கையாளர்களை சந்திக்கவும், உங்கள் கேள்விகளுக்கு பதில் பெறவும்

நீங்கள் சிக்கிக்கொள்வின் அல்லது செயற்கை நுண்ணறிவு முகவர்கள் கட்டமைப்பில் கேள்விகள் இருந்தால், எங்கள் Microsoft Foundry Discord இல் உள்ள ஒதுக்கிய Discord சேனலில் சேரவும்: [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### நீங்கள் தேவைப்படுவது

இந்த பாடநெறியின் ஒவ்வொரு பாடத்திலும் குறியீடு உதாரணங்கள் உள்ளன, அவை code_samples கோப்புறையில் கிடைக்கின்றன. உங்கள் சொந்த நகலை உருவாக்க [இந்த சேமிப்பகத்தை கிளோன் செய்யவும்](https://github.com/microsoft/ai-agents-for-beginners/fork).

இந்த பயிற்சிகளில் உள்ள குறியீடு உதாரணங்கள் Microsoft Agent Framework மற்றும் Azure AI Foundry Agent Service V2 ஐப் பயன்படுத்துகின்றன:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure கணக்கு தேவை

இந்த பாடநெறி Microsoft-இன் கீழ்க்கண்ட AI முகவர் கட்டமைப்புகள் மற்றும் சேவைகளைப் பயன்படுத்துகிறது:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

இந்த பாடநெறிக்கான குறியீட்டை இயக்க மேலதிக தகவலுக்கு [பாடநெறி அமைப்பு](./00-course-setup/README.md) சென்று பார்வையிடவும்.

## பங்களிப்பு

இந்தத் திட்டம் பங்களிப்புகளையும் பரிந்துரைகளையும் வரவேற்கிறது. பெரும்பாலான பங்களிப்புகளுக்கு, நீங்கள் பங்களிப்பை பயன்படுத்தும் உரிமையைக் கொண்டுள்ளீர்கள் என்பதையும், உண்மையில் அதனை வழங்குகிறீர்கள் என்பதையும் விளக்கும்
ஒரு பங்களிப்பாளர் உரிமம் ஒப்பந்தத்திற்கு (CLA) நீங்கள் ஒப்புக்கொள்வது அவசியம். விவரங்களுக்கு <https://cla.opensource.microsoft.com> ஐப் பார்வையிடவும்.

நீங்கள் ஒரு புல் ரிக்வஸ்ட் சமர்ப்பிக்கும் போது, CLA போர்டு உங்களுக்கு CLA அளிக்க வேண்டுமா என்பதை தானாக தீர்மானிக்கிறது மற்றும் PR ஐ சரியான முறையில் அலங்கரிக்கிறது (எ.கா., நிலை சரிபார்ப்பு, கருத்து). போர்டு வழங்கும் வழிமுறைகளைப் பின்பற்றுங்கள். நாங்கள் பயன்படுத்தும் அனைத்து ரெப்போக்களிலும் இதை நீங்கள் ஒருமுறை மட்டுமே செய்ய வேண்டும்.

இந்தத் திட்டம் [Microsoft Open Source Code of Conduct](https://opensource.microsoft.com/codeofconduct/) ஐ ஏற்றுக்கொண்டுள்ளது.
மேலும் தகவலுக்கு [Code of Conduct FAQ](https://opensource.microsoft.com/codeofconduct/faq/) ஐப் பார்க்கவும் அல்லது
